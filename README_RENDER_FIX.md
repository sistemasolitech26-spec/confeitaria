# Render - Confeitaria Olitech V5.2

Build Command:
`npm install --package-lock=false --omit=dev --no-audit --no-fund`

Start Command:
`node server.js`

Depois de subir no GitHub, use no Render:
Manual Deploy > Clear Build Cache & Deploy

No Supabase, execute o `schema.sql` novamente para aplicar as tabelas/colunas novas.
