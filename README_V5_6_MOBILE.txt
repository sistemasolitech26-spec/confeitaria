CONFEITARIA OLITECH - V5.6 MOBILE

Correções aplicadas:
- Layout responsivo para smartphone.
- Menu lateral vira menu recolhível/hambúrguer.
- Corrigido corte da tela para a direita no celular.
- Dashboard, cards e formulários em coluna única no mobile.
- Tabelas com rolagem horizontal controlada, sem quebrar a tela.
- Botões e campos ajustados para toque no celular.
- Badge atualizado para V5.6.

Instalação/atualização:
1. Substitua os arquivos do projeto pelos deste ZIP.
2. Rode npm install se necessário.
3. Rode npm start localmente ou envie para GitHub/Render.
4. No celular, limpe o cache do navegador ou abra em aba anônima para carregar o CSS novo.
